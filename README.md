# Z36s BLE Recon & Control Tool

A Python-based BLE reverse engineering and control shell for DA14583-class smartwatches.

## Features
- Interactive BLE device scanner & selector
- GATT info dumping
- Step / HR / BP / SpO2 queries
- Vibration control
- Raw packet sender
- Probe/bruteforce BLE command space
- Session logging (JSON + TXT)
- Live notification decoding
